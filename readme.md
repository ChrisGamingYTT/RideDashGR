# 🛴 RideDash

**RideDash** is a clean, real-time scooter dashboard app designed for riders who want to see live speed, distance, signal strength, and device status while riding.

Built for Android with a minimal neon-blue HUD interface optimized for quick glances on the road.

---

## 🔵 Features

- ⚡ Live speed tracking (km/h via GPS)
- 📏 Trip distance tracking (km)
- ⏱️ Ride timer
- 📶 Network type indicator (4G / 5G / WiFi)
- 📡 Signal strength (0–4 blue bars)
- 🔋 Battery percentage display
- 🌙 Clean dark HUD design (blue neon theme)

---

## 🎨 UI Concept

RideDash uses a futuristic **blue HUD interface**:

- Large central speed display
- Minimal distractions
- Neon blue glow elements
- Designed for safety and readability while riding

---

## ⚙️ Tech Stack

- Kotlin (Android)
- Jetpack Compose UI
- Android GPS (FusedLocationProviderClient)
- TelephonyManager (network info)
- BatteryManager (battery status)

---

## 🚀 How it works

1. Mount your phone on your scooter
2. Open RideDash
3. Start riding
4. View real-time stats instantly

No setup required — just ride.
